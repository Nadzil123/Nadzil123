## 🚀 Nolqu Ecosystem

The **Nolqu ecosystem** is designed to provide everything needed to build software using Nolqu.

### Core

- **NolquLang** — The programming language
- **Nolqu VM** — Bytecode virtual machine
- **Nolqu Compiler** — Compiles Nolqu code

### Tools

- **Nolqu CLI** — Command line tools
- **Nolqu Playground** — Online code runner
- **Nolqu Formatter** — Code formatting tools

### Libraries

- **Nolqu Standard Library**
- **Nolqu Modules**

### Documentation

- Language documentation
- Tutorials
- Examples
